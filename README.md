# Supply Chain Risk Detection System

An end-to-end machine learning pipeline for detecting anomalies and predicting 
risk in logistics and supply chain data.

## Models used
- Isolation Forest — unsupervised anomaly detection
- Random Forest — risk tier classification (Low / Medium / High)
- SHAP — model explainability

## Notebook structure
1. Data loading
2. Feature engineering
3. Anomaly detection
4. Risk classification
5. Model evaluation and calibration
6. SHAP explanation
7. Business insights and export

## Requirements
pandas, numpy, scikit-learn, matplotlib, seaborn, shap
